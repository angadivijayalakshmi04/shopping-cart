A backend API for an online shopping cart system built using Spring Boot, JPA, MySQL, and Swagger.
This project is a RESTful Shopping Cart API that supports product management, cart operations, order placement, and email notifications.
Built using Spring Boot 2.7.2, it provides clean, scalable, and production-ready API endpoints for any e-commerce application.
The API includes Swagger UI documentation, making it easy to test and integrate with frontend or mobile applications.
